# ⚠️ DEPRECATED - Moved to jis-core

This repository has been **renamed and moved** to:

## 👉 [jaspertvdm/jis-core](https://github.com/jaspertvdm/jis-core)

### Why the rename?
The `did:jis` prefix conflicted with W3C DID specification. We've pivoted to `jis:` URIs as a standalone identity standard.

### Migration
```bash
# Old (deprecated)
pip install did-jis-core

# New
pip install jis-core
```
